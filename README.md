# Frontend Mentor - Typing Speed Test App

![Design preview](./desktop-started.jpg)

## Overview

This is my solution to the **Typing Speed Test App** challenge on Frontend Mentor. The application allows users to practice typing while tracking their speed and accuracy with real-time feedback.

## Features

- Start typing by clicking the **Start** button or simply typing.
- Choose between three difficulty levels:
  - Easy
  - Medium
  - Hard
- Random passage generation from a local JSON file.
- Real-time typing feedback:
  - ✅ Correct characters
  - ❌ Incorrect characters
  - Cursor highlighting
- Live statistics:
  - Words Per Minute (WPM)
  - Accuracy
  - Timer
- Restart the test at any time.
- Result screen after completing the test.
- Personal Best saved using **Local Storage**.
- Special screens:
  - Baseline Established
  - High Score Smashed
- Responsive design for desktop and mobile devices.
